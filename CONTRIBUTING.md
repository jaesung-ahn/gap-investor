# Contributing to Gap Investor

Welcome to the Gap Investor project! This document provides context and guidelines for contributors, including AI agents.

## Project Context
- **Type**: Spring Boot Application (Java)
- **Build System**: Gradle
- **Key Directories**:
  - `src/main/java`: Source code
  - `.agent/workflows`: AI-specific workflows

## Development Guidelines
- Please follow standard Java coding conventions.
- Ensure `./gradlew test` passes before submitting changes.

## For AI Agents
- Check `.agent/workflows` for specific tasks like deployment or onboarding.
- Read this file to understand general architectural decisions.
